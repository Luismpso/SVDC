## 🎯 Course Objectives

Throughout the semester, this repository will document my progress and learning in the following core topics:

1. **Introduction to Quantitative Data Visualization:** Fundamentals and basic principles.
2. **Theories, Processes, and Models:** Understanding how to effectively represent data visually.
3. **Web Visualization Development:** Creating and implementing interactive visualizations in a web environment.
4. **Visual Data Analysis:** Techniques for exploring and interpreting datasets.
5. **Knowledge Extraction:** Using visualization platforms to generate valuable insights.
6. **Effectiveness Evaluation:** Validating visualizations according to specific tasks and target audiences.
